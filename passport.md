## 4. Физическая модель и приближения

### 4.1. Основная физическая модель

Кратко описать, какие уравнения, сечения, модели или физические формулы лежат в основе кода.

```text
Код вычисляет число фотонов обратного комптоновского рассеяния в однофотонном приближении (линейный режим), интегрируя локальное дифференциальное сечение по пересечению электронного и лазерного пучков.
Дифференциальное сечение (вероятность излучения) берётся в виде, пропорциональном формуле Клейна-Нишины для неполяризованных электронов, умноженной на фактор линейной поляризации фотона:
dN/(dΩ dE) ~ ∫ d^3r dt n_e(r,t) n_l(r,t) * (dσ/dΩ) * (энергетический фактор).
В коде это реализовано через предварительный расчёт интеграла пересечения I(θ_x, θ_y) (intersection) как функции углов электронов, а затем свёртка с энергетическим распределением электронов и дифференциальным сечением.
Энергия рассеянного фотона задаётся параметром s = E_γ / (γ m c^2) и связана с углом наблюдения через соотношение s = (γ m c^2) / (1 + (γθ)^2) в ультрарелятивистском пределе.
Эффект нелинейности эмулируется сдвигом γ0 → γ0 / sqrt(1 + a0^2/8) и уширением σ_γ за счёт a0^4.
```

### 4.2. Как устроен расчет

Кратко описать расчетный процесс (workflow) без деталей реализации.

```text
1. Задать параметры электронного сгустка (заряд, эмиттансы, размеры) и лазерного импульса (энергия, длина волны, перетяжка, длительность).
2. Вычислить пересечение (intersection): дискретизировать углы электронов (θ_x, θ_y) и методом Монте-Карло с частицами-сэмплами получить двумерный массив I(θ_x, θ_y), представляющий интеграл перекрытия плотностей электронного и лазерного пучков в пространственно-временном объёме.
3. Для расчёта полного выхода: просуммировать I(θ_x, θ_y) с весовыми множителями углового распределения электронов.
4. Для спектра: проинтегрировать I(θ_x, θ_y) по углам, свернуть с гауссовым энергетическим распределением и дифференциальным сечением.
5. Для спектрально-углового распределения: для каждой точки (θ_x, θ_y, s) на сетке выполнить интегрирование по кольцевым секторам (arcs) внутри эллиптического детектора, используя дополнительный Монте-Карло слой для учёта углового разрешения и особенностей сечения.
```

### 4.3. Основные приближения

```text
1. Линейное комптоновское рассеяние (однофотонное взаимодействие), нелинейные эффекты эмулируются феноменологическим сдвигом и уширением.
2. Ультрарелятивистский предел (γ >> 1) для связи энергии и угла рассеяния.
3. Гауссовы профили электронного сгустка (в координатах и по энергии) и лазерного импульса (поперечный профиль ТЕМ00, продольная огибающая гауссова).
4. Параксиальное приближение для лазерного пучка.
5. Отсутствие обратного влияния излучения на динамику электронов (не учитывается радиационное трение).
6. Пренебрежение многократным рассеянием.
7. Малые углы рассеяния (θ_x, θ_y) – разложение sinθ ≈ θ, cosθ ≈ 1.
```

### 4.4. Что принципиально не входит в модель

```text
1. Излучение на гармониках
2. Отдача/множественное рассеяние
```

## 6. Область применимости

### 6.1. Проверенные режимы

```text
1. Линейный режим, a0 = 0 (или emulate_nonlinearity=False). Расчёты полного выхода и спектров визуально правдоподобны, но сравнение с аналитикой не зафиксировано.
2. Эллиптические электронные пучки (sigma_ex != sigma_ey), астигматические пучки (emittance_x != emittance_y)
```

### 6.2. Режимы, требующие проверки

```text
1. Слабо-нелинейный режим
2. Смещение фокусов (delta_x, delta_y, delta_z ≠ 0).
```

### 6.3. Режимы, где код не следует использовать без доработки

```text
1. Сильные поля (a0 > 1), где доминирует нелинейное комптоновское рассеяние и генерация гармоник.
2. Область параметров, где энергия фотонов сравнима с энергией электронов (глубокий квантовый режим) – формула сечения может быть некорректна.
```
